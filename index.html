09:00 Thiago Scalone

Git, O que é? Do que é feito? Para que serve?

Apesar de estarmos em 2016, muitos programadores não entendem o que é Git, ou o que ele faz, apenas decoram comandos para executarem no final do dia de trabalho. Vamos mudar isso! Vamos conversar sobre Git focando na prática, entre o iniciante e avançado. Discutir o conceito por trás do Git e controles de versão distribuídos, assim como fluxos de trabalho, boas e más praticas e comandos úteis.

10:00 Cassio Thadeu

Docker 1.12, Orchestration made easy

A utilização de Docker para deploy de microservices já não é nenhuma novidade. Com ele é possível alcançar vários benefícios da arquitetura baseada em microservices, mas a partir da release 1.12 sua relação com containers Docker irá mudar, a partir de agora orquestração, escalonamento, multi-host nerworking, service discovery e load balancing já é uma realidade da engine padrão, facilitando a vida dos DevOps e aumentando a disponibilidade dos ambientes de produção. Nessa talk iremos conhecer as novidades por tráz dessa nova versão do Docker, seu funcionamento interno, montagem de cluster de Docker e orchestração de containers. Talk is cheap. Show me the code
